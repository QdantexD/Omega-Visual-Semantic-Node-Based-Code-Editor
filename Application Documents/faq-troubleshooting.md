# FAQ & Troubleshooting (EN)

## Terminal node does not show the console
- Verify the node type is `Terminal`.
- Ensure you opened the editor with a double-click.
- Check for initialization errors in the selected profile.

## Git Bash does not appear
- Install Git for Windows and ensure `bash.exe` is on PATH.

## Terminal changes are not saved
- Use `Ctrl+S` in the node content editor window.
- Confirm the project is saved from the main window.

## Shortcuts do not respond
- Check focus: some shortcuts require focus on the editor or canvas.
- Review conflicts with OS/system shortcuts.

## Build fails with PyInstaller
- Verify dependencies in `requirements.txt`.
- Confirm `CodemindEditor.spec` points to `version-1.5.0-alpha.txt`.
- Run in PowerShell with appropriate permissions.

## How to report issues
- Include version, terminal profile, steps to reproduce, and logs if possible.
