# Controls & Shortcuts (EN)

## Canvas (Blueprint Editor)
- Select: click a node; drag to move.
- Zoom: mouse wheel or controls in the bottom bar.
- Node context menu: right-click a node.
- Double-click a node: open the node’s content editor.
- ESC: leave node edit mode and return to the canvas.

## Node Content Editor
- Save content: `Ctrl+S`.
- Embedded Terminal:
  - Top bar: choose profile (`PowerShell`, `CMD`, `Git Bash`) and close terminal.
  - Command input: type and press Enter.
  - Copy/paste: use OS shortcuts (`Ctrl+C`, `Ctrl+V`).

## Connections
- Create a connection: drag from an output port to a compatible input port.
- Remove a connection: via the link’s context menu or node actions.

## Inspection & Panels
- Node Inspector: view/edit properties.
- Realtime Variables panel: observe and update state.
- Code Map: quick view of artifacts.

## Tips
- Terminal nodes are minimal outside edit mode (title only) to reduce visual clutter.
- Use different terminal profiles per node to scope tasks.
- Save regularly to persist terminal transcription and graph changes.
